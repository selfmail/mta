# MTA

This is the official Selfmail **Mail Transfer Agent**. It's an open-source alternative to services like [Haraka](https://haraka.github.io) or [Postalserver](https://postalserver.io). It's ment to be used to send and receive emails on your own server, with a polished nice user interface.

## Progress

This project is currently in development and is not expected to be ready for production anytime soon.
